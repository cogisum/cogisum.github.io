发表于 2018-05-06

测试网络带宽

方法一：使用speedtest.net

直接执行./speedtest-cli会选择就近的服务器进行速度测试，如果要指定某个位置的服务器，可以使用--list参数列出可用的服务器，然后在执行的时候添加--server <SERVER>选项即可。
通过这种方法可以分别给出上传和下载速度。

方法二：使用wget

这样会给出下载时间和平均下载速度。
