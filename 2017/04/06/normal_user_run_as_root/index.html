如何以普通用户执行root权限

发表于 2017-04-06

背景

有一个程序希望能够执行一些需要root权限的操作，但是又不想每次都输入密码。

问题

要解决上述问题，一种可行的方案是解决这么一个问题：如何以普通用户的身份执行root操作？比如passwd程序

方法

可以通过setuid或者setgid的方法。以setuid为例，它允许用户以文件owner的权限运行。这样如果文件owner是root，并且其他用户有执行权限，那么任何普通用户都可以以root权限执行该文件。
但是注意该方法存在安全风险，且在Linux下对解释性脚本无效，见参考链接。
